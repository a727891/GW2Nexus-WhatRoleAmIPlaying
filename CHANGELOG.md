# Changelog

All notable changes to the Nexus **What Role Am I Playing?** addon are documented here.

## [Unreleased]

### Added

- Initial Nexus port of the BlishHUD What Role Am I Playing module
- Random raid role picker with DPS / boon DPS / healer filters
- Main ImGui window and corner icon with context menu
- `roles.json` sync from `bhm.blishhud.com` with `last_updated` cache metadata
- gw2dat icon downloads cached under addon `textures/assets/`
- Identifiable `User-Agent` (`WhatAmIPlaying-Nexus/1.0.0`)

### Notes

- Elite spec unlock filtering matches Blish development mode (all specs treated as unlocked)
- GW2 API account/character integration is planned for a future release
